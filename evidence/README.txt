TRUSTA v5 증거 이미지 폴더
==========================

이 폴더에 아래 파일명으로 이미지를 넣으면, 버전5(?v=5) 페이지의
해당 evidence-card에 자동으로 표시됩니다. (없으면 "증거 이미지 자리" 표시)

필요한 파일:
- mba.png      : MBA 졸업장 (이미 적용됨 · 개인정보 블러 처리본)
- fb-ad.png    : Facebook/광고 성과 화면 (민감정보 블러 권장)
- calendar.png : 월간 운영 캘린더 실제 화면
- ui.png       : 상세/랜딩 Before·After UI 화면

규칙:
- 원본은 잘리지 않고(object-fit: contain) 전체가 보존됩니다.
- 비율이 안 맞으면 같은 이미지를 흐리게 깔아 배경을 채웁니다.
- 계정명/이메일/캠페인명/QR/서명 등 민감정보는 넣기 전에 블러 처리하세요.

교체 후에는 다시 배포해야 라이브에 반영됩니다.
